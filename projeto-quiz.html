Quiz de Análise de Pele

Stack: React, TypeScript, Vision API
Ano: 2025

O Problema

Escolher produtos de skincare online é um desafio. Sem a orientação de um profissional, consumidores costumam comprar itens inadequados para seu tipo de pele, resultando em insatisfação e desperdício.

A Solução

Desenvolvi uma interface interativa que vai além de um simples formulário. O projeto **realiza uma captura de imagem em tempo real** da pessoa através da câmera, identificando zonas críticas e texturas, combinando esses dados visuais com as respostas do quiz.

Como Funciona

O sistema processa a imagem para detectar características faciais. Com base no cruzamento das informações coletadas (oleosidade, sensibilidade, idade) e na análise visual do rosto, o algoritmo recomenda uma rotina de produtos específica para aquele usuário, aumentando a precisão da compra e a confiança na marca.
