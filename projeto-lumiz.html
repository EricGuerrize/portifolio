Lumiz — Gestão Clínica

Stack: React, Banco de Dados, TS
Ano: 2025
Ver Site ao Vivo ↗

O Problema

Clínicas médicas muitas vezes sofrem com sistemas de gestão arcaicos, lentos e pouco intuitivos, o que resulta em atrasos no atendimento, perda de dados e dificuldade no acompanhamento histórico dos pacientes.

A Solução

O Lumiz foi desenvolvido para ser uma plataforma centralizada e de alta performance. Implementei um sistema de agendamento em tempo real, gestão de prontuários eletrônicos e integração direta com banco de dados relacional para garantir integridade e velocidade.

Como Funciona

Utilizando React no frontend para uma experiência fluida (SPA) e Supabase no backend para autenticação e persistência de dados. O sistema permite que médicos e secretários gerenciem toda a jornada do paciente, desde o primeiro contato até o histórico de consultas e prescrições, tudo em uma interface limpa e moderna.
