Auditoria de Transparência

Stack: Python, Tkinter, API
Ano: 2024

O Problema

A fiscalização de pagamentos públicos e benefícios sociais costuma ser um processo manual e demorado. Auditores precisam cruzar listas extensas de CPFs com portais de transparência, o que é inviável para grandes datasets.

A Solução

Desenvolvi uma ferramenta desktop em Python que automatiza esse cruzamento. O sistema consome APIs oficiais para verificar a situação de beneficiários e funcionários públicos, gerando alertas imediatos sobre possíveis irregularidades.

Como Funciona

A aplicação utiliza Tkinter para uma interface leve e funcional. Implementei um sistema de caching local para otimizar as chamadas de API, garantindo que a auditoria seja rápida mesmo com milhares de registros. Os dados processados podem ser exportados diretamente para Excel com relatórios prontos para uso oficial.
